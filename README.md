## Fullstack-Movie

Fullstack Movie App, Node.js ile kendi backend’iniz kullanılarak geliştirilmiş bir film sitesi olup, film arama, kategori filtreleme ve film ekleme/çıkarma özelliklerini sunar.
Veri JSON formatında saklanır, frontend HTML, CSS, Tailwind ve Vite ile hazırlanmıştır ve tüm CRUD işlemleri backend tarafından yönetilmektedir.
